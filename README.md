## Installation & Usage

### 1. Clone the repository
```bash
git clone https://github.com/math-12393/Autistic-Skating-Group.git
cd Autistic-Skating-Group
```

### 2. View the website locally
Open `index.html` in any modern web browser:

```bash
open index.html   # macOS
xdg-open index.html   # Linux
start index.html   # Windows
```

### 3. Access the live GitHub Pages site
The repository is automatically published via **GitHub Pages**. Visit:

**https://math-12393.github.io/Autistic-Skating-Group/**

### 4. Repository URL
All source code, media, and documentation are hosted at:

**https://github.com/math-12393/Autistic-Skating-Group**
